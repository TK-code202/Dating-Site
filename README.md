# _Celebrity Dating Site_

#### _This website matches users with celebrities based on their traits._

#### By _**Tofunmi Kusimo**_

## Technologies Used

* _GIT_
* _HTML5_
* _CSS_
* _BOOTSTRAP 4.5.3_
* _Javascript and jQuery_


## Description

_The website gathers input from the users and matches them with celebrities according to the information inputted by the user using branching._

## Setup/Installation Requirements

* _Clone this repository to your desktop_
* _Navigate to the top level of the directory_
* _Open the index.html file in your browser._
